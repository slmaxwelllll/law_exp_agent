  - 同一范畴的主题归入一组（如「解除婚姻关系」和「解除婚姻关系（无民事行为能力人）」属于同组）

    将其标记为「离群组」且 label 以「[离群]」开头，不参与最终模板归纳
输出JSON：{"groups": [{"label": "组的简短标签", "indices": [1,3,5]}, ...]}
indices 对应该组包含的 case 的 index 列表。
